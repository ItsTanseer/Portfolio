# Tanseer Ahmad's Portfolio Website

A modern, visually stunning portfolio website showcasing projects, skills, and professional expertise. Built with React, Tailwind CSS, and custom WebGL animations.

**Link**: 

---

## 🎨 Overview

This portfolio is a web application designed to create an immersive, interactive experience that highlights:
- **Captivating UI/UX** with custom animations and effects
- **Multiple showcase sections** (Home, Skills, Projects, Contact)
- **Responsive design** optimized for mobile and desktop
- **WebGL-powered background effects** for stunning visuals
- **Smooth animations** using AOS (Animate On Scroll) and Motion libraries

The design philosophy centers on creating a memorable first impression while maintaining professional credibility through clean typography and organized information architecture.

---

## ✨ Key Features

### 1. **Dynamic Background (SoftAurora Component)**
- Custom WebGL shader-based aurora animation
- Real-time Perlin noise implementation
- Mouse-interactive effects
- Configurable colors, speed, and visual parameters
- Smooth gradient transitions with customizable brightness

### 2. **Smooth Animations**
- **AOS (Animate On Scroll)**: Elements animate as they enter the viewport
- **Shiny Text Effect**: Custom text shine/gradient animation using Motion library
- **Hover States**: Interactive micro-interactions on images and buttons
- **Staggered Reveals**: Cards and sections appear with controlled timing

### 3. **Responsive Layout**
- Mobile-first design using Tailwind CSS
- Adaptive grid layouts
- Touch-friendly navigation
- Optimized for screens from 320px to 4K

### 4. **Interactive Skill Carousel**
- Auto-scrolling skill logos
- Mask gradient overlay for smooth edge transitions
- Hover scale effects
- Smooth performance with CSS animations

### 5. **Project Showcase**
- Card-based layout with hover effects
- Shadow and glow effects on interaction
- Quick links to view all projects
- Project descriptions with visual previews

### 6. **Contact Integration**
- Direct social media links (Instagram, LinkedIn, GitHub)
- Email contact method
- Interactive button states
- Professional call-to-action

## 🛠 Tech Stack

### Frontend Framework
- **React 18+** - Component-based UI
- **React Router** - Client-side navigation
- **Tailwind CSS** - Utility-first styling

### Animation & Effects
- **AOS (Animate On Scroll)** - Scroll trigger animations
- **Motion (Framer Motion)** - Advanced animation primitives
- **Custom WebGL** - Aurora background shader effects
- **OGL (Open Game Library)** - WebGL rendering

### Build & Tooling
- **Vite** or **Create React App** (specify your setup)
- **Node.js** - Development environment
